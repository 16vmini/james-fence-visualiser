# James's Front Wall — Fence Visualiser

An interactive mock-up of wood fencing on top of the curved white front wall.
Open `index.html` (or the GitHub Pages link) and play with fence style, colour,
height, board width, gap and top line. Everything is in one self-contained HTML
file with the photo embedded.

- **Hold to see original** — press and hold to compare before/after.
- **Adjust fit** — drag the handles if the fence doesn't sit exactly on the wall.
